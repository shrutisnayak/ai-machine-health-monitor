# 🛠️ AI-Powered Machine Health Monitor

An interactive dashboard that simulates industrial machine monitoring using AI and sensor data.

---

## 🚀 Features

- 🔁 Real-time sensor simulation (Temperature + Vibration)
- 🧠 AI-based anomaly detection using Isolation Forest
- ⚠️ Smart alert system (Normal / Warning / Critical)
- 📈 Trend detection (early failure signals)
- 📊 Historical analytics dashboard
- 📥 Downloadable machine health report (CSV)
